# curso-de-graphql
Curso de GraphQL

Proyecto de GraphQL

Servidor en nodejs + GraphQL

Para iniciar:

```
git clone git@github.com:DevcodeInc/curso-de-graphql.git
npm install
npm run db:migrate
npm run db:seed
npm run start
```
